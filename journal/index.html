What I Can Improve

Supplemental Learning

I still enjoy going old school and watching tutorials however it is just so much faster and accurate and useful to use an LLM these days. I am taking this class so that I am forced to read and write PHP myself so I can better understand autogenerated boilerplate code and tweak as necessary.

Week 2

Topics Covered

  • Control Structure and Loops
  • Web Hosting

Quick Summary

We did not meet in class during Week Two. The topics we covered included using PHP to create control structures such as loops. We were also asked to compare and contrast different web hosting platforms and evaluate whether they satisfy the type of hosting environment we will need in order to work with PHP and SQL.

What Felt Easy / Challenging

  • Logic is always difficult even when it's basic. I am thankful I have a background in JavaScript already so I have experience with loops and counters. I found the discussion about choosing a web platform much easier. The question seems a bit outdated because these days most hosting providers are fully capable of running SQL and PHP in their updated iterations. I've been using Siteground for several years now so it's an easy choice for me!

Aha Moments

  • It was nice to compare different hosting providers because I want to learn to create websites for clients and I am out of touch with pricing because I have been using Siteground for so long now but it looks like they're all about the same; they start cheap and then after a year they jack up the price! It doesn't seem to matter which hosting provider you go with in that regard.

Connections to Other Courses

The obvious connection off of the top of my head is the parallels between PHP and JavaScript. It seems like they handle logic in pretty much the same manner. I've heard that if you can learn the syntax and structure of one coding language it somewhat transferable to other languages. I think that's cool that I'm experiencing that firsthand!

What I Like Best About My Work

It was fun using PHP logic to calculate discounts automatically.

What I Can Improve

There is always more to learn. Syntax will always be an issue. And learning the cleanest flow of logic.

Supplemental Learning

  • I just finished building a personal website for a Black Sabbath tribute band that I'm in this week. Part of that was implementing an admin page to update gigs for the gig calendar. I used PHP to accept form data and render it to the admin page and the user page simultaneously
